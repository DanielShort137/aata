Abstract Algebra: Theory and Applications

Everything you wanted to know about abstract algebra, but were afraid to buy

From the links below you can download a PDF version of Abstract Algebra: Theory and Applications and the corresponding LaTeX source which is licensed under the GFDL.

The 2009 PDF version below is very close to the original text, as first published by PWS Publishing Company in 1994. Since then, the copyright has transferred back to the author, making this open-source version possible. A full copy of the copyright transfer is distributed with the source code in the local subdirectory. Current work will fix a variety of errors and modernize the latex code - a new version based on these changes should be available here in very late August 2010.

Latest Versions, with Revision Control

The most recent version is available through a Mercurial repository at bitbucket.org, the specific address is http://bitbucket.org/judson/aata. Links above to source code archives should retrieve the very latest versions available at bitbucket.org. If you would like to participate in contributing to the book, you will want to use Mercurial. Then you can create a local copy of the source code by doing the following in a terminal window:
  1. Set your working directory to a place where a new directory named aata can live.
  2. Issue the command: hg clone https://judson@bitbucket.org/judson/aata
If you would like to make substantial contributions to the book, sending a Mercurial patch is welcome. If you wish to make substantial contributions, or create an alternate version, please save yourself some trouble and contact the author beforehand.
