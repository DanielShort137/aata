From the links below you can download a PDF version of Abstract Algebra: Theory and Applications and the corresponding LaTeX source which is licensed under the GFDL.

The PDF version below is close to the original, as first published by PWS Publishing Company in 1994. Since then, the copyright has transferred back to the author, making this open-source version possible. A full copy of the copyright transfer is distributed with the source code in the local subdirectory. Current work (Summer 2010) will fix a variety of errors and modernize the latex code.

Latest Versions, with Revision Control

The most recent version is available through a Mercurial repository at bitbucket.org, the specific address is http://bitbucket.org/judson/aata. "Current Version" links above will create archives of this repository. If you would like to participate in contributing to the book, you will want to use Mercurial. Then you can create a local copy of the source code by doing the following in a terminal window:
  1. Set your working directory to a place where a new directory named aata can live.
  2. Issue the command: hg clone https://judson@bitbucket.org/judson/aata
If you would like to make substantial contributions to the book, sending a Mercurial patch is welcome. If you wish to make substantial contributions, or create an alternate version, please save yourself some trouble and contact the author beforehand.
